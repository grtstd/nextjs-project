# Todo Site

A clean todo website built with **Next.js** and **Bun**.

## Features

- Add new tasks
- Mark tasks as done
- Delete tasks
- Filter by **all / active / completed**
- Clear all completed tasks
- Simple dark UI

## Tech

- Next.js
- React
- TypeScript
- Tailwind CSS
- Bun

## Run locally

```bash
bun install
bun run dev
```

Then open:

```bash
http://localhost:3000
```

## Project structure

- `app/page.tsx` — main todo UI
- `app/layout.tsx` — layout + metadata
- `app/globals.css` — global styles
